# Contributing to RosCard

Thank you for helping improve RosCard!

---

## Ways to Contribute

- Bug reports
- Feature requests
- Documentation updates
- Code improvements

---

## Development Setup

1. Fork repository
2. Clone your fork

git clone https://github.com/yyqclhy/RosCard

3. Install dependencies

npm install

4. Start development build

npm run dev

---

## Pull Request Rules

- Keep PR focused
- Describe problem solved
- Include screenshots if UI changes
- Follow existing code style

---

## Code Philosophy

Home Assistant handles logic.
RosCard handles interaction.
Astrion handles experience.
